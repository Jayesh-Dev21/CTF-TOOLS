CTF Solver's Toolkit

Websites

  • CTFtime

    A central hub for CTF enthusiasts, providing event schedules, team rankings, and writeups from past competitions. Use it to study solutions, track upcoming challenges, or join a team.

  • CyberChef

    A versatile web tool from GCHQ for encoding, decoding, encrypting, and data manipulation. Ideal for quick cryptography tasks—like base64 decoding or XOR operations—without writing code.

  • dCode

    A collection of online solvers for ciphers (e.g., Caesar, Vigenère, RSA) and tools for hash cracking or frequency analysis. Perfect for identifying and breaking obscure encodings in crypto challenges.

  • Exploit-DB

    A searchable archive of exploits and proof-of-concept code. Great for finding known vulnerabilities in software encountered during binary or web exploitation tasks.

  • HackTricks

    An extensive, regularly updated guide to hacking techniques, tools, and CTF strategies. Covers everything from privilege escalation to steganography—use it as a quick reference during challenges.

  • Aperi’Solve

    An online steganography analyzer that runs tools like zsteg, steghide, and binwalk on uploaded images. Upload a suspicious image to extract hidden data or flags effortlessly.

  • Cryptii

    A modular web tool for encoding, decoding, and encrypting text. Its pipeline interface lets you chain operations (e.g., ROT13 then base64), making it handy for layered crypto puzzles.

  • Hashcat

    The homepage for Hashcat, a high-performance password cracker. Download it to tackle hash-cracking challenges (e.g., MD5, SHA256) with GPU acceleration.

  • Pwnable.xyz

    A platform offering binary exploitation and reverse engineering challenges. Practice ROP, buffer overflows, and more in a competitive environment.

  • OverTheWire

    Wargames with hands-on challenges for Linux skills, binary exploitation, and networking. Start with the Bandit series to build foundational skills for CTF.

  • TryHackMe

    An interactive platform with CTF-style rooms for web, network, and forensics practice. Offers guided learning paths alongside standalone challenges.

  • CrackStation

    A free online hash cracker for common types like MD5 and SHA1. Use it for quick lookups when you suspect a hash is from a known password list.

  • StegOnline

    A web-based steganography tool for analyzing and modifying images. Extract hidden data or tweak LSB values directly in your browser.

  • CTFlearn

    A beginner-to-advanced platform with community-submitted challenges and writeups. Great for practicing diverse skills and learning from others’ solutions.

  • ROPgadget

    A tool (and site) for finding ROP gadgets in binaries. Download the script to automate gadget discovery for exploitation in pwn challenges.

Python Tools

  • Python Learning Resources
  • Pwntools

    A CTF framework for exploit development, especially binary exploitation. Simplifies tasks like crafting payloads, interacting with remote services, and building ROP chains (pip install pwntools).

  • Ciphey

    An AI-driven tool that automatically identifies and decrypts encodings, ciphers, and hashes. Feed it garbled text, and it’ll crack it for you (pip install ciphey).

  • angr

    A binary analysis framework using symbolic execution to explore program paths. Use it to solve reverse engineering puzzles or find flags in complex binaries (pip install angr).

  • pycryptodome

    A robust library for cryptographic operations (e.g., AES, RSA, SHA). Write custom scripts to encrypt, decrypt, or analyze crypto challenges (pip install pycryptodome).

  • hashpumpy

    A tool for hash length extension attacks on algorithms like MD5 or SHA1. Exploit vulnerable hash-based authentication in web challenges (pip install hashpumpy).

  • rsatool

    A script to calculate RSA private keys from public parameters (e.g., n, e). Essential for RSA-based crypto challenges with weak keys (clone repo).

  • xortool

    A tool to analyze and crack multi-byte XOR ciphers by guessing key lengths and keys. Perfect for XOR-encrypted flags (pip install xortool).

  • sqlmap

    An automated tool for SQL injection attacks. Point it at a vulnerable web app to extract databases or bypass logins (clone repo).

  • Padding Oracle Exploit

    A script to exploit padding oracle vulnerabilities in encryption. Decrypt ciphertexts in crypto challenges with misconfigured padding (clone repo).

  • requests

    A simple HTTP library for web exploitation. Craft custom requests, scrape pages, or automate interactions with web challenges (pip install requests).

  • binwalk

    A tool for analyzing and extracting data from binary files (e.g., firmware). Use it in forensics to uncover hidden files or flags (pip install binwalk).

  • Pillow

    A library for image manipulation. Write scripts to analyze or modify images for steganography challenges (pip install pillow).

  • stegano

    A Python library for LSB steganography. Hide or extract data in images programmatically (pip install stegano).

  • unicorn

    A CPU emulator for running and debugging binary code. Emulate instructions to understand obfuscated programs (pip install unicorn).

  • capstone

    A disassembly framework for reverse engineering. Parse binary instructions for analysis or scripting (pip install capstone).

  • python-nmap

    A Python wrapper for Nmap. Automate network scanning in challenges requiring port enumeration (pip install python-nmap).

  • scapy

    A packet manipulation tool for crafting, sending, and analyzing network traffic. Forge packets or sniff data in networking tasks (pip install scapy).

  • pyinstaller-unpacker

    A script to unpack PyInstaller-compiled Python executables. Reverse engineer packed binaries to find hidden logic (clone repo).

  • BeautifulSoup

    A library for parsing HTML/XML. Scrape web pages or extract data from messy source code (pip install beautifulsoup4).

  • z3-solver

    A theorem prover for solving constraints. Use it for symbolic execution or cracking logic-based challenges (pip install z3-solver).
